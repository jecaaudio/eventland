# Eventland

Pagina web inicial para vender servicios de organizacion, decoracion y produccion de eventos, con sonido e iluminacion proporcionados por JECA Audio.

## Servicios incluidos

- Organizacion de eventos
- Decoracion y ambientacion
- Iluminacion profesional
- Sonido por JECA Audio
- Produccion tecnica
- Fotografia

## Archivos principales

- `index.html`: estructura de la pagina
- `styles.css`: estilos visuales y responsive
